# Cross-cohort prognostic analysis of breast cancer gene signatures

This repository contains R scripts used for the analysis of PAM50 and candidate prognostic gene signatures across TCGA-BRCA and METABRIC breast cancer cohorts.

## Datasets
- TCGA-BRCA
- METABRIC

## Analyses performed
- PAM50 scoring
- PCA
- Kaplan–Meier survival analysis
- Cox proportional hazards modelling
- Candidate gene signature derivation

## Software
- R
- TCGAbiolinks
- SummarizedExperiment
- survival
- survminer
- ggplot2
